enum AXPTZInvoke

The type of invoke to be used by functions in the PTZ movement and preset handlers.

Supported type: AX_PTZ_INVOKE_ASYNC with callback function and userdata set to NULL. Invokes and executes the PTZ movement operation. The type is asynchronous; the invoke will not wait until the execution is finished.
