Detailed Description

This is the main include file to be used for all applications that want to use the AXSerialPort library.

Purpose

Use the AXSerialPort library to communicate with external hardware using the serial port found on selected products.

Important

The AXSerialPort library is designed to be used with the GLib library. An application using the axserialport library must have a running GMainLoop.

◆ AXSerialParity

enum AXSerialParity

The parity bit. A parity bit is added to make sure that the number of bits with the value one in a set of bits is even or odd.

None = No parity bit is added. Even = The parity bit is set to 1 if the number of ones in a given set if bits (including the parity bit) is odd. Odd = The parity bit is set to 1 if the number of ones in a given set if bits (including the parity bit) is even. Mark = The parity bit is always 1. Space = The parity bit is always 0.

Enumerator
AX_SERIAL_PARITY_NONE 

No parity bit is added

AX_SERIAL_PARITY_EVEN 

The parity bit is set to 1 if the number of ones in a given set if bits (not including the parity bit) is odd.

AX_SERIAL_PARITY_ODD 

The parity bit is set to 1 if the number of ones in a given set of bits (not including the parity bit) is even.

AX_SERIAL_PARITY_MARK 

The parity bit is always 1.

AX_SERIAL_PARITY_SPACE 

The parity bit is always 0.
