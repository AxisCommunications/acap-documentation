Detailed Description

This is the main include file to be used for all applications that want to use the axstorage library.

Purpose

The AXStorage library enables applications to save and retrieve data on mounted storage devices such as SD cards and NAS (Network Attached Storage) units. An application can create files on the storage device and can modify and remove its own files.

The following features are supported:

List available storage devices. Get the path to the application's files on the storage device. The path is used by the application to access the files (read, write, modify). Subscribe/unsubscribe to events (available, exiting, writable, full).

Important

The axstorage library is designed to be used with the GLib library. An application using the axstorage library must have a running GMainLoop.
