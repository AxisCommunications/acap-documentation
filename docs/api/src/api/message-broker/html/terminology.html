Terminology
Name Description
channel An object that publishes messages on a topic and source.
client A producer or consumer of messages.
consumer An application that owns one or more subscriber objects.
info Channel information, e.g calibration tables and data structure.
message Opaque data structure which contains the payload sent by a producer on a channel.
monitor An object that allows for finding channels and reading their info.
payload The data published by producers on channel objects, and received by consumers through subscriber objects.
producer An application that owns one or more channel objects.
source A topic can be divided into several sources. A channel is uniquely identified by the combination of a topic and a source. Access control is enforced on the topic level.
subscriber An object that receives messages on a topic and source.
topic A textual string that identifies the messages sent/received. Access control is configured on the topic level.
